# HamadanGPT

HamadanGPT is a chatbot designed to answer questions with up-to-date information and document-based input.

## Features

- Responds to **recent and current events**
- Accepts and processes uploaded files:
  - **PDF** (`.pdf`)
  - **Text** (`.txt`)
  - **Images** (`.jpg`, `.png`)
- Can analyze content from files and answer questions based on them

## Use Cases

- Asking questions about current news or events  
- Uploading documents or images for explanation, summary, or analysis  
- General-purpose chatbot assistance
